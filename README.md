# Haze Haus 🧶

Welcome to **Haze Haus**, a premium, interactive web catalog showcasing handmade crochet creations crafted with love and passion by Samrudhi and Mom.

Explore cozy blankets, amigurumi, stylish apparel, and unique accessories through a responsive, animation-rich user interface.

## 🚀 Key Features

- **Splash Screen Animation:** A theme-oriented stitching needle & thread opening transition.
- **Dynamic Catalog:** Interactive filtering by categories: *Amigurumi*, *Apparel*, and *Accessories*.
- **Quick View Modal:** In-depth product descriptions and options to add to cart or buy now.
- **Local Cart Drawer:** Sidebar shopping cart drawer displaying item previews, pricing, and live total calculation.
- **Interactive Search:** Full overlay search functionality with real-time matching and text highlighting.
- **Custom Orders:** Direct links to Instagram/email for personalized handmade commissions.

---

## 📁 Repository Structure

All codebase files are located in the root of the repository:

- **[`index.html`](index.html):** The primary landing page featuring the splash screen, hero showcase, features, testimonials, product grids, and contact section.
- **[`purchase.html`](purchase.html):** A template page handling checkout and order placement.
- **[`style.css`](style.css):** Tailored styles implementing elegant color palettes, typography, responsive layouts, hover effects, and modern CSS transitions.
- **[`script.js`](script.js):** Client-side interactivity including product inventory arrays, catalog filtering, cart state management, search logic, and UI modal controls.
- **[`.gitignore`](.gitignore):** Configured to ignore temporary system files and editor settings.
- **Media Assets:** Asset showcase files containing product photography (`*.jpg`, `*.png`, `*.svg`) and video demos (`*.mp4`).

## 🛠️ How to Run Locally

Since Haze Haus is built using modern core web technologies (HTML5, Vanilla CSS3, and JavaScript), it runs entirely client-side without any complex build steps.

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/rustedpotato/Hazehaus.git
   cd Hazehaus
   ```
2. **Open the Project:**
   - Double-click [`index.html`](index.html) to run it in your default web browser.
   - Or, run it with an editor-based local server like **VS Code Live Server** for optimal resource loading.
